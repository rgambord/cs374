Reserved Words and Operators

We have already come across several operators, such as the redirection operators, the pipe (|) operator, and the separators ; and &. An additional set of reserved words and control operators are used to implement control flow in the shell.

Reserved Words

Several control flow constructs used in shell scripting rely on the use of reserved words, or keywords. There are sixteen reserved words,

!

do

esac

in

{

done

fi

then

}

elif

for

until

case

else

if

while

These words are only recognized as reserved words when none of the characters are quoted and when the word is:

  • The first word of a command (i.e. without any preceding redirections or assignment words)

  • The first word following another reserved word other than case, for, and in

  • The third word in a case command (only in is syntactically valid in this case)

  • The third word in a for command (only in and do are valid in this case)

Note

Reserved words are fairly intuitive and can essentially be thought of special builtin commands–perhaps this gives a hint as to how they are actually implemented.

The ones that tend to confuse students are the !, {, and } reserved words. Unlike operators, these are subject to the above recognition rules, and must appear as independent words. For example, !command attempts to run a command named !command, while ! command logically negates the exit value of command. This is more an accident of history rather than a thoughtful design choice.

Control Operators

The remaining constructs rely on the control operators, some of which we have already been introduced to. There are a total of eight control operators:

&

|

;

&&

||

;;

(

)

Note

The major difference between reserved words and operators is that reserved words are only recognized in specific situations, and when set apart from surrounding words with whitespace. On the other hand, operators are recognized wherever they appear as long as they are unquoted. Thus, echo hello&echo world is identical to echo hello & echo world
